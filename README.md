**HR Analytics Dashboard**

This repository contains an interactive HR Analytics Dashboard built using Power BI to analyze workforce metrics 
such as attrition, tenure, salary distribution, demographics, and performance trends. 
The project showcases end-to-end data analytics skills including cleaning, modeling, DAX calculations, and visualization.

**Project Objectives**
Understand overall employee distribution and demographics
Analyze Attrition %, Tenure, and Salary Insights
Identify workforce trends across departments and cities
Evaluate employee performance scores and satisfaction
Support HR decisions with actionable insights
